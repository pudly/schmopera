=== Using the Theme Customizer ===

Typable is customizable via the WordPress customizer. Here, you can upload a logo, set your accent color and add custom CSS.

* To access the theme customizer, click Appearance -> Customize in the WordPress admin menu.

* You should now be in a live preview of Typable, with the customizer on the left.

* Use the Typable Styling menu to customize your site with a logo, accent colors and custom CSS.

* When you're finished making changes, click Save & Publish to save the settings.

* Check out your site to confirm your changes.


=== Post Styles ===

Typable comes with a few custom element styles, which are used to easily add extra styling to your WordPress posts. To use the post styles, simply select your text and then select from the Styles drop down (on the post editor) which style you would like to apply.

Intro Title - Select any text and select Page Title from the drop down to add a styled block of text to introduce your page. Similar to blockquote styling.

Highlight - Select any text and select Highlight from the styles drop down to add a yellow highlight style to your post.
